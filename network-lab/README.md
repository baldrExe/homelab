# Network Lab — Pratique CompTIA Network+

Lab personnel pour pratiquer les concepts réseau,
en préparation à la certification CompTIA Network+.

## Progression

| Thème | Statut | Concepts couverts |
|-------|--------|-------------------|
| [Subnetting](./subnetting/) | 🔄 En cours | CIDR, masques, calcul d'hôtes |
| [Protocoles](./protocols/) | ⏳ Planifié | TCP/IP, DNS, DHCP, HTTP/S |
| [Topologies](./topologies/) | ⏳ Planifié | LAN, WAN, schémas Packet Tracer |

## Objectif

Maîtriser le subnetting de tête (sans calculatrice),
comprendre le modèle OSI et TCP/IP en profondeur,
et être capable de diagnostiquer des problèmes réseau
de manière méthodique.

## Outils utilisés

- Cisco Packet Tracer (gratuit avec compte Cisco NetAcad)
- ipcalc (Linux) pour vérifier les calculs
- subnettingpractice.com pour l'entraînement chronométré
